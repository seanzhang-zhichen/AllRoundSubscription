## 如何查看图表

这些文档中包含多个使用Mermaid语法编写的图表。查看这些图表的方式有：

1. 在支持Mermaid渲染的Markdown查看器中查看（如GitHub、GitLab等）
2. 复制Mermaid代码到[Mermaid Live Editor](https://mermaid.live)中查看和导出
3. 使用Visual Studio Code的Mermaid插件本地查看
